Инструкция по работе с приложением "Конвертер запросов СФР"

Для начала работы с приложением проверьте статус о готовности к работе. Если указано, что требуется обновить базу данных в появившемся диалоговом окне, нажмите "да". После изменения статуса на "готов к работе" можете приступить к формированию запроса.

1. Выберите необходимый справочник запросов.

2. Выберите необходимые параметры запроса.

3. Введите фамилию и иницалы человека в определённые поля ввода. После ввода фамилии и инициалов кнопка "Преобразовать" станет активной.

4. Если необходимо указать напоминание, установите флажок. Напоминание нужно для того, если запрос был сделан ранее и на него не было ответа.

5. Чтобы сформировать запрос нажмите на кнопку "Преобразовать".

6. Откроется диалоговое окно "Выберите файл для конвертирования", в котором нужно выбрать необходимый файл для конвертирования. После чего нажмите кнопку "Открыть".

7. Появится диалоговое окно "Выберите директорию куда сохранить файл", в котором нужно выбрать небходимую директорию для сохранения сконвертированного запроса. Нажмите кнопку "Выбор папки".

8. Появится сообщение о том, что ваш запрос успешно сконвертирован. Нажмите Ок. Ваш файл в формате "***.zip" появился в указанной директории.

Формирование запросов в ДНР, ЛНР, Запорожскую и Херсонскую области

Для того, чтобы сформировать запросы в ДНР, ЛНР, Запорожскую и Херсонскую области нужно:


1. Выбрать необходимый справочник.

2. Выбрать ОСФР выступающее посредником и такое же значение выбрать в списке УСФР. После этого в интерфейсе приложения появится список с выбором суффикса вышеуказанных регионов.

3. Заполните поля ФИО и укажите необходимый суффикс региона. После чего нажмите "Преобразовать".

4. Далее повторите пункты 6, 7 раздела инструкции по работе с приложением "Конвертер запросов СФР". После чего файл с именем "*******.zip" появится в указанной вами директории.

Вопросы и ответы. FAQ

1. В статусе указано "Требуется обновить базу данных", значит то, что были внесены изменения в коды запросов и результат выполнения запроса может быть неверным. Если появилось всплывающее окно с сообщением об обновлении, нажмите "Да".

2. Если всплывающее окно не появилось, нажмите "Файл", а потом "Получить обновление БД". После чего статус изменится на "Готов к работе". Можете приступать к формированию запроса.

3. В статусе указано "Отсутствует подключение к базе данных", значит то, что корректность кодов запросов не гарантируется. Для решения этой проблемы обратитесь к сотруднику, ответственному за работу приложения в вашем подразделении.
