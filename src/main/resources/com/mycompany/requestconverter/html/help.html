Инструкция по работе с приложением "Конвертор запросов ПФР"

Для начала работы с приложением проверьте статус о готовности к работе. Если указано, что требуется обновить базу данных в появившемся диалоговом окне, нажмите "да". После изменения статуса на "готов к работе" можете приступить к формированию запроса.

1. Выберите необходимые параметры запроса.

2. Введите фамилию и иницалы человека в определённые поля ввода. После ввода фамилии и инициалов кнопка "Преобразовать" станет активной.

3. Если необходимо указать напоминание, установите флажок.

4. Чтобы сформировать запрос нажмите на кнопку "Преобразовать".

5. Откроется диалоговое окно "Выберите файл для конвертирования", в котором нужно выбрать необходимый файл для конвертирования. После чего нажмите кнопку "Открыть".

6. Появится диалоговое окно "Выберите директорию куда сохранить файл", в котором нужно выбрать небходимую директорию для сохранения сконвертированного запроса. Нажмите кнопку "Выбор папки".

7. Появится сообщение о том, что ваш запрос успешно сконвертирован. Нажмите Ок. Ваш файл в формате "***.zip" появился в указанной директории.

Вопросы и ответы. FAQ

1. В статусе указано "Требуется обновить базу данных", значит то, что были внесены изменения в коды запросов и результат выполнения запроса может быть неверным. Если появилось всплывающее окно с сообщением об обновлении, нажмите "Да".

2. Если всплывающее окно не появилось, нажмите "Файл", а потом "Получить обновление БД". После чего статус изменится на "Готов к работе". Можете приступать к формированию запроса.

3. В статусе указано "Отсутствует подключение к базе данных", значит то, что корректность кодов запросов не гарантируется. Обратитесь к администратору баз данных для решения этой проблемы.
